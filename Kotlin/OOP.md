# 객체지향 프로그래밍이란. 
   - Object Oriented Programming (OOP)
   - 코틀린은 모든 것이 클래스 형태이므로 객체화할 수 있다. 
   - 프로그램에서 필요한 데이터를 추상화시켜 상태와 행위를 가진 객체를 만든다.
   - 객체들간의 적절한 결합을 통해 유지 보수를 쉽게 한다.
   - 5대 키워드 (클래스, 추상화, 캡슐화, 상속, 다형성)
